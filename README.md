GST Calculator

A simple and user-friendly GST Calculator built using HTML, CSS, and JavaScript.

Overview

This tool helps users quickly calculate the GST amount and the final payable amount based on the selected GST rate. It is designed with a clean interface and provides instant results.

Features

- Enter any amount to calculate GST
- Choose GST rates: 5%, 12%, 18%, or 28%
- Instant calculation of GST amount
- Displays the final amount including GST
- Simple and responsive user interface
- Built using pure HTML, CSS, and JavaScript

Technologies Used

- HTML
- CSS
- JavaScript

How to Use

1. Enter the amount.
2. Select the GST rate.
3. Click the Calculate GST button.
4. View the GST amount and total payable amount instantly.
